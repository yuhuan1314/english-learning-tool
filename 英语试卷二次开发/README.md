# 📝 英语试卷二次开发工具

一个基于Web的英语试卷二次开发工具，帮助教师将普通试卷转化为深度学习材料。

## 功能特点

- 📤 PDF试卷上传与解析
- 🔍 自动识别题型（阅读理解、七选五、完形填空、语法填空等）
- 📚 自动生成二次开发练习
  - 话题词汇（动作引擎、语义支柱、修饰词）
  - 词性变形练习
  - 短语翻译
  - 语境填词
  - 语篇结构分析
  - 微写作
- 💾 导出Word文档

## 安装运行

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
streamlit run app.py
```

### 3. 打开浏览器
浏览器访问: http://localhost:8501

## 使用方法

1. 在侧边栏选择要处理的题型
2. 选择要包含的练习类型
3. 上传PDF试卷
4. 点击"生成二次开发练习"
5. 下载生成的Word文档

## 技术栈

- **前端**: Streamlit
- **PDF处理**: PyPDF2
- **文档生成**: python-docx
- **数据分析**: pandas
